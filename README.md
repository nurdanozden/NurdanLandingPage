## ✨ Öne Çıkan Özellikler

| | |
|---|---|
| 🎨 **Pastel Pembe Tasarım** | Çok katmanlı radial + linear gradient ile göz yormayan, şık bir arka plan |
| 🕊️ **Animasyonlu Kuşlar** | CSS keyframe + SVG ile 18 kuş uçuşu — her biri farklı hız, boyut ve yön |
| 💌 **Mektup Bölümü** | Framer Motion stagger animasyonlarıyla açılan, handwriting fontlu kişisel bir not |
| 🖼️ **Galeri & Fotoğraf** | Sonsuz marquee galeri + Swiper.js ile tam ekran fotoğraf carousel'i |
| ✦ **Sayfa Geçişleri** | `whileInView` ve `viewport` ile scroll-driven Framer Motion animasyonları |
| 📱 **Tam Responsive** | Mobil-first Tailwind utility sınıflarıyla her ekranda mükemmel görünüm |

---

## 🛠️ Teknoloji Stack'i

```
React 19       →  UI bileşenleri
Tailwind CSS 4 →  Utility-first stilleme (@theme ile custom token'lar)
Framer Motion  →  Scroll & enter animasyonları
Swiper.js      →  Dokunmatik destekli fotoğraf carousel'i
Vite 7         →  Hızlı geliştirme ve optimize production build
```

---

## 🚀 Çalıştırmak İçin

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Production build al
npm run build
```
